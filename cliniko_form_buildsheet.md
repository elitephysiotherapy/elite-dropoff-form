# Cliniko Pre-Assessment Form - Build Sheet

Work straight down this list in Cliniko's form builder. Section 1 (About you) is
already done. This covers Sections 2-12.

## How to use it

- **New section:** click **Add section**, type the Section title, and (where shown)
  the Section description.
- **Each question:** click **Add question**, type the Question title exactly as
  written, set **Choose type**, set **Required**, and for Multiple choice /
  Checkboxes add each option on its own line.
- **Save often:** click **Save template** every section or two so nothing is lost.
- Type names are Cliniko's: Single line text, Paragraph text, Multiple choice,
  Checkboxes, Date, Signature.
- Only the questions marked **Required: Yes** need it on - everything else is No.
